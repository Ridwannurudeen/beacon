Need bUSD?

Beacon USD (bUSD) is an open-mint EIP-3009 token on X Layer testnet. Mint any amount to yourself by calling mint(yourAddress, amount) on bUSD. Each unit has 6 decimals — pass 100000000 for 100 bUSD.

Withdraw semantics: ERC-4626 withdrawals come from the vault's idle bUSD balance. For larger redemptions when capital is allocated to strategies, use the WithdrawQueue (request → admin recall → claim).
